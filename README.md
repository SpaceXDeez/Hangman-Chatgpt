# Hangman

A browser-based hangman game built with HTML, CSS, and JavaScript.

## Run the game

1. Open `index.html` in your browser.
2. Start guessing letters using your keyboard or the on-screen keys.
3. Use the "New Word" button to restart anytime.

## Notes

- The game tracks up to 6 misses before revealing the answer.
- The "Reveal Word" button shows the answer and ends the round.
